# Run with Docker
```bash
# Quickly run a copy
docker run menendezpoo/fragment:latest \
       -e FRAGMENT_DATA_DRIVER=sqlite \
       -e FRAGMENT_ROOT_PASSWORD=root \ 
       -p 80:80    
```
